# Discord議事録Bot

DiscordのVCで喋ったことをチャットに文字起こしします。

## 機能

- スレッドチャンネルを自動作成してその中に文字起こしします。
- Google Speech-to-Text APIを使用しています

## 使用方法

- VCに入って `/recording start` で書き起こしを開始します
- `/recording stop` で終了します
